# django-channels-example
Django Channels

##### Install
Windows:
```
1 - virtualenv env
2 - env\Scripts\activate.bat
3 - pip install -r requirements.txt
```
Mac / Linux
```
1 - python3 -m venv env
2 - source env/bin/activate
3 - pip install -r requirements.txt
```

##### Requirements
```
- Redis
- Python3
```
